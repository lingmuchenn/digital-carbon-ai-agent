# 数字脱碳管家 - 用户使用指南

## 🚀 快速开始

### 1. 首次配置（必须）

1. 找到 `config.json.example` 文件
2. 复制并重命名为 `config.json`
3. 用文本编辑器打开，填入您的 API 密钥

```json
{
  "DASHSCOPE_API_KEY": "sk-你的通义千问密钥",
  "DEEPSEEK_API_KEY": "sk-你的DeepSeek密钥"
}
```

### 2. 运行应用

- **Windows**: 双击 `数字脱碳管家.exe`
- **Mac**: 双击 `数字脱碳管家.app`

### 3. 使用功能

1. 点击右下角的悬浮球图标
2. 选择分析模式（相似文件/重复文件）
3. 输入要分析的文件夹完整路径
4. 点击"开始分析"
5. 等待分析完成
6. 在卡片中处理结果

## 📁 路径格式

### Windows
```
D:\文件夹\子文件夹
C:\Users\用户名\Documents
```

### Mac
```
/Users/用户名/Desktop/文件夹
/Users/用户名/Documents/项目
~/Downloads/资料
```

## ⚠️ 首次运行可能遇到的问题

### Windows
**提示**: "Windows 已保护你的电脑"
- 点击 "更多信息"
- 点击 "仍要运行"

### Mac
**提示**: "无法打开，因为无法验证开发者"
- 右键点击应用
- 选择 "打开"
- 在对话框中点击 "打开"

或使用终端命令：
```bash
xattr -cr 数字脱碳管家.app
```

## 💡 使用技巧

### 后台运行
- 点击空白处收起悬浮球
- 分析会在后台继续
- 再次点击查看进度

### 批量处理
- 勾选要删除的文件
- 点击"确认全部"处理所有
- 点击"跳过全部"忽略所有

### 配置调整
修改 `config.json` 可以：
- 更换 API Keys
- 调整相似度阈值
- 更换 AI 模型

## 🆘 常见问题

**Q: API 调用失败？**
- 检查 API Keys 是否正确
- 检查账户余额
- 确认网络连接

**Q: 找不到文件夹？**
- 确保路径存在
- 检查访问权限
- 使用绝对路径

**Q: 分析很慢？**
- 文件多时需要时间
- 请耐心等待
- 可以在后台运行

**Q: 如何卸载？**
- Windows: 直接删除文件夹
- Mac: 拖到废纸篓

## 🔧 系统要求
